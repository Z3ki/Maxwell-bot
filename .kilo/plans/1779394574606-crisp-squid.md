# Telegram Connection + Better Auto Mode Plan

## Objective
1. Add a Telegram connection to talk directly to Maxwell, ask it questions, and receive responses.
2. Improve the `auto_decider_prompt` for auto-reply mode to make it sharper, funnier, and more robust.

## Design Decisions

### 1. Telegram Connection Architecture
- **Integration**:
  - Store `TELEGRAM_TOKEN` in `.env` (never commit tokens).
  - Add a background polling task in `bot.py`.
  - When a message is received on Telegram:
    - Build LLM context.
    - Query the LLM provider.
    - Reply back asynchronously.
  - Limit Telegram access to configured admin IDs only.

### 2. Better Auto-Mode Decider Prompt
Make the decider prompt sharper to avoid "AI slop" or ignoring interesting conversations.

## Verification Tasks
- Send a Telegram DM to the bot: verify it responds as Maxwell.
- Check bot logs via PM2 to ensure no polling errors.
- Verify the auto mode decider prompt has been updated.
